# Guess The Word (Insider & Chameleon)

A mobile-friendly, pass-and-play web application featuring two popular spoken word party games: **Insider** and **Chameleon**. Designed to be played in a group passing a single phone around.

## Games Included

### 1. Chameleon
* **The Premise:** Everyone in the group knows the secret word except one person—the Chameleon.
* **The Gameplay:** Everyone goes around the room saying a single related word. The Chameleon must guess a word to blend in.
* **The End:** The group votes on who the Chameleon is. If caught, the Chameleon gets one final chance to guess the secret word.

### 2. Insider
* **The Premise:** Only one person knows the secret word—the Insider.
* **The Gameplay:** The group discusses and asks questions to guess the word before the timer runs out. The Insider secretly tries to guide the group toward the correct word.
* **The End:** If the word is guessed, the group votes on who the Insider is. If the Insider stays hidden, the Insider wins!

## How to Run Locally

Since this is a client-only Single Page Application (SPA), you can run it without any installation:
1. Clone or download the repository.
2. Open `index.html` in any web browser.
